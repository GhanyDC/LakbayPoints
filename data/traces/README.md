# Generated GPS Trace Data

These files are deterministic generated prototype traces for the final Phase
0A Cubao-to-Hulo multimodal route:

- `valid_phase_0a_multimodal_trace.json`
- `suspicious_phase_0a_multimodal_trace.json`

The valid trace covers public-road movement, MRT and ferry dwell/transfers, MRT
and ferry movement, and both required walking legs. The suspicious trace covers
the same chain but contains an impossible off-route teleport.

They mirror the shared TypeScript exports and are validated by automated tests.
They are not collected real trips, pilot evidence, live GPS, or proof of field
accuracy.
